### Getting Started

#### Install dependencies

```bash
# use yarn(recommend)
$ yarn
# or if using npm:
$ npm i
```

#### Run dev server

```bash
$ yarn run dev
# or if using npm:
$ npm run dev
```

### Generate static files

```bash
$ yarn run build
# or if using npm:
$ npm run build
```
